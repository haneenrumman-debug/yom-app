# # Yōm — يوم 🌟
> مساعدك الصوتي الذكي لإدارة الوقت وتحقيق الإنجاز — مثل Siri بس بالعربي

---

## 📌 نبذة عن المشروع

**Yōm** مش تطبيق مهام عادي — هو رفيق ذكي يبادرك ويسألك ويتعلم منك.
- Siri وAlexa تنتظرك تسألها → **Yōm هو يحكيك**
- يرصد عادات تسويفك ويبني خريطة يومك
- يعدّل جدولك تلقائياً ويوصيك بما يناسبك

## ⚠️ قرارات تقنية مهمة

- **لماذا Flutter؟** → فريق صغير، كود واحد لمنصتين
- **لماذا Railway وليس AWS؟** → أرخص وأسهل للمبتدئين
- **لماذا Firebase؟** → مجاني لمرحلة MVP والإعداد سريع
- **لماذا Android أولاً؟** → Apple Store يكلف $99/سنة + Mac

## 🚀 كيف تشغّل المشروع

```bash
# 1. clone المشروع
git clone https://github.com/your-username/yom.git

# 2. ادخل المجلد
cd yom

# 3. ثبّت الاعتماديات
flutter pub get

# 4. شغّل التطبيق
flutter run
```

## 🔑 متغيرات البيئة المطلوبة

أنشئ ملف `.env` في جذر المشروع:

```
CLAUDE_API_KEY=your_claude_api_key
WHISPER_API_KEY=your_openai_api_key
FIREBASE_PROJECT_ID=your_firebase_project_id
```

> ⚠️ لا ترفع ملف `.env` على GitHub أبداً — أضفه لـ `.gitignore`
